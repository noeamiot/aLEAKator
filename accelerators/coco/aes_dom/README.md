# Configurable order DOM implementation of AES Accelerator

Original version [github repository](https://github.com/hgrosz/aes-dom), detailed in
[Domain-Oriented Masking: Compact Masked Hardware Implementations with Arbitrary Protection Order](https://eprint.iacr.org/2016/486.pdf).

The version used as submodule in this repository is this [github repository](https://github.com/vedadux/aes-dom/).
The only change is that default protection order is set to 1.
